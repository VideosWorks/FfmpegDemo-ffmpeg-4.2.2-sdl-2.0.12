# FfmpegDemo


    a project with ffmpeg 4.2.2 , sdl 2.0.12 for playing video


-------

* #### 简介
    这是一个关于集成ffmpeg的demo，用于学习简单的ffmpeg的demo编写

* #### 目前技术框架
    相关框架：ffmpeg sdl

* #### 项目构建流程

    >**[🔗 NO.5 FFmpeg 4.2.2 集成](https://blog.csdn.net/qq_38315348/article/details/105487438)**

* #### 项目的功能

    * ffmpeg -> 视频每一帧转为YUV420P
    * sdl -> 播放YUV420P的视频
